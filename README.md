Firmware del progetto per il Laboratorio di Sistemi Embedded dell'Università degli Studi della Campania "Luigi Vanvitelli".

Il progetto è stato realizzato da:
- Gennaro Lepore
- Pietro Paolo Buonomo
- Francesco Di Marzo

Il dispositivo si compone di un microcontrollore Arduino Due ed un circuito composto di un amplificatore operazionale allo scopo di adattare il segnale in ingresso ai valori di tensione e corrente supportati dall' ingresso del microcontrollore
